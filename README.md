### web-create - 前端项目脚手架

#### 安装

```bash
npm install web-create -g
```

#### 使用

```bash
web-cli create <project-name>
```

#### 更新

#### 问题

1. 模板项目地址来源于 github，下载的时候可能需要翻墙，如果下载失败，请重新尝试。
2. github 开放 api 调用次数有限制。
